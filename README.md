# Pomodoro
Productivity App – Pomodoro Timer & Task Manager

SwiftUI alapú iOS alkalmazás, amely segít hatékonyabban beosztani az idődet egy Pomodoro időzítő és egy prioritásalapú feladatkezelő segítségével. Kizárólag iPhone-ra optimalizálva, modern MVVM architektúrában készült.

Funkciók:
- Pomodoro Timer - Teljes képernyős visszaszámláló animációval, munka- és pihenőidővel, értesítésekkel.
- Feladatkezelő - Prioritás alapján rendezett to-do lista. Feladatok pipálhatók, szerkeszthetők, törölhetők.
- Helyalapú Feladatok - Feladatokhoz térképes helyszín rendelhető MapKit segítségével.
- Feladat részletei - Teljes információ a feladatról, időtartam, helyszín és státusz megjelenítés.
- Widget támogatás - Gyors hozzáférés az időzítőhöz vagy a legfontosabb feladatokhoz a kezdőképernyőről.

Tech Stack
- SwiftUI + MVVM
- Core Data (perzisztens tárolás)
- MapKit & CoreLocation
- UserNotifications
- WidgetKit
